# Controle de LED com Bluetooth

Desenvolver uma aplicação prática de controle remoto para acender e apagar um LED utilizando comunicação Bluetooth. O objetivo principal é que o aluno aprenda a configurar e programar um módulo Bluetooth para que ele se comunique com um microcontrolador (como Arduino) e controle o estado de um LED, reforçando conceitos de eletrônica básica, programação de dispositivos embarcados e comunicação sem fio.
